# imdb_scrapper  
**INFO**  
Simple movies and series imdb scrapper  
Used modules requests, BeautifulSoup4, lxml
  
**HOW TO RUN**  
python3 -m venv venv    
. venv/bin/activate    
pip install -r requirements.txt    
python3 imdb_scrapper.py
